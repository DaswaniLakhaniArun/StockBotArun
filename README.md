# StockBot

Bot financiero en Telegram que envía:

- Resumen diario de mercado  
- Alertas automáticas cuando un activo sube/baja más del umbral  
- Detección de apertura y cierre de la bolsa  
- Seguimiento intradía de tus acciones favoritas  
- 24/7 en un servidor como Render  

---

## Características

- Datos desde *yfinance*  
- Alertas configurables  
- Mensajes programados  
- Fácil despliegue  
- Código modular  
- Perfecto para portfolio o CV
